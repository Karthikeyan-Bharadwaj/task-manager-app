# Task Manager Application

A full-stack Task Manager Application built using React, Node.js, Express, MongoDB Atlas, and JWT Authentication.

## Live Demo

Frontend:
https://task-manager-app-five-ivory.vercel.app

Backend API:
https://task-manager-app-1hkf.onrender.com


## Features

* User Registration
* User Login
* JWT Authentication
* Create Tasks
* View Tasks
* Update Tasks
* Delete Tasks

## Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT
* bcryptjs

## Run Backend

```bash
cd Backend
npm install
npm run dev
```

## Run Frontend

```bash
cd Frontend
npm install
npm run dev
```
